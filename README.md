
# AI Factuality Detection

## Project Overview

This project focuses on detecting whether AI-generated text is factual or non-factual.
With the growing use of generative AI systems, ensuring that responses contain reliable and accurate information is increasingly important.
The goal of this project is to analyze AI-generated outputs and classify them based on factual correctness using data analysis and machine learning techniques.

## Dataset

The dataset used in this project contains AI-generated responses labeled according to their factual accuracy.
The data is stored inside the **data/** folder and is used for model training and evaluation.

---

## Methodology

The project workflow includes the following steps:

1. Data preprocessing and cleaning
2. Exploratory Data Analysis (EDA)
3. Model training and validation
4. Prediction generation
5. Evaluation of model performance

All experiments and analysis are performed inside the **notebook.ipynb** file.
